## ğŸ® How to Play

### Controls
- **WASD** - Move your character
- **E** - Call emergency meeting
- **K** - Kill (Impostors only)
- **V** - Vent (Impostors only)
- **ESC** - Return to lobby
- **Mouse** - Click UI buttons

### Game Flow
1. **Lobby** - Select player count (4-15) and impostor count (1-3)
2. **Game Starts** - You're assigned a random role
3. **Playing** - Complete tasks (as Crewmate) or kill players (as Impostor)
4. **Voting** - Emergency meetings to vote out suspicious players
5. **Game End** - Impostors win if they equal/outnumber crewmates, or Crewmates win if tasks are done/impostors ejected

### Your Role

**Crewmate:**
- Move around and complete tasks
- Discuss who the impostor is
- Vote to eject suspicious players

**Impostor:**
- Pretend to do tasks while observing
- Kill crewmates (25-second cooldown)
- Use vents to escape (10-second cooldown)
- Avoid being voted out
